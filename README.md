# Password Manager

A secure **Password Manager web application** built with **Next.js**, enabling users to safely store, manage, and organize their passwords.

---

## 🚀 Features

- Store, edit, and delete passwords securely
- Responsive design for desktop and mobile
- Fast performance with server-side rendering (SSR)

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS  
- **Backend:** Next.js API Routes, Node.js    

---
